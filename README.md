# contextual-semantic-file-sorter-csfs

A Python utility project.

## Installation

```bash
git clone https://github.com/Manas236/contextual-semantic-file-sorter-csfs
cd contextual-semantic-file-sorter-csfs
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
